## üé® Tailwind Medical Theme Setup

### 1. **Edit `tailwind.config.js`**

Add custom colors for your hospital palette:

```js
// tailwind.config.js
module.exports = {
  content: ["./src/**/*.{js,jsx,ts,tsx}"],
  theme: {
    extend: {
      colors: {
        primary: {
          DEFAULT: "#2563EB", // Blue
          dark: "#1D4ED8",
          light: "#E0F2FE",
        },
        secondary: {
          DEFAULT: "#16A34A", // Green
          light: "#86EFAC",
        },
        background: {
          DEFAULT: "#F9FAFB", // Light background
          dark: "#F3F4F6",
        },
        text: {
          DEFAULT: "#111827", // Dark gray for text
          muted: "#4B5563",   // Subtle gray
        },
        error: "#DC2626",     // Red for critical alerts
        warning: "#F59E0B",   // Amber for warnings
        info: "#0891B2",      // Teal for info
      },
    },
  },
  plugins: [],
};
```

### 2. **Usage Examples in Components**

Now you can use these semantic classes across your app:


### 3. **Recommended Component Styling**

* **Cards / Sections:** `bg-white shadow-md rounded-xl p-6`
* **Navigation Bar:** `bg-primary text-white`
* **Progress Steps (Registration ‚Üí Tests ‚Üí Results ‚Üí Report):** Use `bg-primary` for active, `bg-gray-200` for inactive
* **Tables (for lab results):** `border border-gray-200 text-text bg-white`
